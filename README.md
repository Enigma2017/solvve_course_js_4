# solvve_course_js_4
It is a project to find lost and found animals. Its mission is to help people find their pets.

## Project Setup

```sh
npm run setup
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run release
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
